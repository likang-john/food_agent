# AI饮食助手

## 项目简介
AI饮食助手是一个集成了食物查询、卡路里计算和当地美食推荐功能的智能饮食顾问系统。该系统通过前后端分离架构实现，帮助用户做出更健康的饮食选择。

## 功能特性

### 后端功能
- **聊天交互**：支持自然语言查询，智能识别用户意图
- **食物查询**：提供常见食物的营养信息查询接口（正在开发）
- **卡路里计算**：返回食物的详细卡路里和营养成分数据（正在开发）
- **美食推荐**：根据用户位置提供当地特色美食推荐（正在开发）

### 前端功能
- **实时聊天界面**：直观的用户交互体验
- **营养信息展示**：以卡片形式展示食物的卡路里和营养成分
- **推荐列表展示**：美观的推荐餐厅和美食展示
- **响应式设计**：适配不同设备屏幕

## 技术栈

### 后端
- **框架**：Flask 3.0.0
- **跨域支持**：Flask-CORS 4.0.0
- **运行环境**：Python 3.10+

### 前端
- **框架**：React 19.2.0
- **构建工具**：Vite
- **样式**：CSS3
- **运行环境**：Node.js 18+

## 安装部署

### 前提条件
- Python 3.10+
- Node.js 18

### 手动安装配置

#### 后端安装与配置
1. 确保已安装Python 3.10或更高版本：
   ```bash
   python --version
   ```
2. 进入后端目录：
   ```bash
   cd backend
   ```
3. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
4. 启动后端服务：
   ```bash
   python main.py
   ```
   服务将运行在 http://localhost:5000

#### 前端安装与配置
1. 确保已安装Node.js 18或更高版本：
   ```bash
   node --version
   npm --version
   ```
2. 进入前端目录：
   ```bash
   cd frontend
   ```
3. 安装npm依赖：
   ```bash
   npm install
   ```
4. 启动React开发服务器：
   ```bash
   npm run dev
   ```
